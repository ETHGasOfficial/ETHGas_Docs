# Obol Validator

## API Endpoints

<div className="api-endpoints-grid">

### Register Obol operator

<details className="api-endpoint">
<summary className="api-endpoint-header">
  <span className="api-method-post">**POST**</span> `/v1/user/obol/operator/register` - Register Obol operator
</summary>

**Code Example:**
<Tabs>
<TabItem value="http" label="HTTP" default>

```bash
curl -H "Authorization: Bearer {{access_token}}" -X POST /v1/user/obol/operator/register \
  -d '{"ownerAddress":"0x1234","operatorId":1,"operatorPublicKey":"0xabc","signature":"0xsig"}'
```

</TabItem>
<TabItem value="python" label="Python">

```python
import requests

url = "https://mainnet.app.ethgas.com/api/v1/user/obol/operator/register"

payload = {
    "ownerAddress": "0x1234",
    "operatorId": 1,
    "operatorPublicKey": "0xabc",
    "signature": "0xsig"
}

headers = {
  "Content-Type": "application/json",
  "Authorization": "Bearer {{access_token}}"
}

response = requests.post(url, headers=headers, json=payload)

print(response.text)
```

</TabItem>
</Tabs>


Request

| Parameter | Required | Type | Description |
| --- | --- | --- | --- |
| ownerAddress | YES | String | Owner address for the Obol operator |
| operatorId | YES | Integer | Unique operator identifier |
| operatorPublicKey | YES | String | Obol operator public key |
| signature | YES | String | Signature authorizing the registration |


Response Body

| Name | Type | Description |
| --- | --- | --- |
| success | boolean | Operation success status |
| operator | object | Registered operator object |

</details>
